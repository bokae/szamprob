Nyári élmény szófelhő¶

Ahhoz hogy lefusson az alábbi notebook telepíteni kell a wordcloud csomagot. Ezt egy terminálban a következő képp tehetjük meg:

pip install wordcloud --user

Ha a telepítés megvan akkor importáljuk a szüksges könyvtárakat:

In [1]:
%pylab inline
import json
from wordcloud import WordCloud

Populating the interactive namespace from numpy and matplotlib

Az adott könyvtárba gyűjtöttem az élmény beszámolós notebookokat, ezeknek a neveit beletöltöm a files listába:

In [2]:
#getting the list of files
files=!ls |grep -v get_what|grep ipynb

Kiválogatom a megfelelő (potenciálisan abeszámolókat tartalmazó) cellákat és minden sort egy listába foglalok

In [3]:
# getting the first markdown cell
dat=[]
for f in files:
    nb=open(f);
    nb=json.load(nb);
    dat.append(nb['cells'][0]['source'])
    if len(nb['cells'])>1:
        dat.append(nb['cells'][1]['source'])
flatdat=[item for sublist in dat for item in sublist];

Kiszűröm a táblázatos, címsoros, linkes és csupán sortörést tartalmazó sorokat (a | és # valamint a http illetve \n karakterekre szűrve)

In [4]:
words=[]
for line in flatdat:
    if not('#' in line) and not('|' in line) and not('http' in line) and not(line=='\n'):
        words.append((line.replace('*','')).replace('-',''))

A hasznos sorokat egy nagy karakterláncba gyűjtöm

In [5]:
bigstr=(''.join(words)).lower()

Ez csak kozmetika egy olyan függvény ami minden szót zöldeskékre szinez. Ezt fogjuk használni a WordCloud függvény egy kulcsszavas argumentumában

In [6]:
def grey_color_func(word, font_size, position, orientation, random_state=None, **kwargs):
    #return "hsl(0, 0%%, %d%%)" % random.randint(60, 100)
    return "rgb(0, 100, 100)"

A szófelhő legyártása itt történik. A stopwords kulcsszavas argumentummal specifikálhatjuk hogy melyik szavakat nem szeretnénk látni a felhőben.

In [7]:
wordcloud = WordCloud(stopwords={'de','ez','ezt','voltam','volt','nyáron','az','a','is','emellett','és','egy'},
                      background_color='white',
                      mode='RGB',
                      width=640, height=480,
                      color_func=grey_color_func,
                      prefer_horizontal=0.5).generate(bigstr)
